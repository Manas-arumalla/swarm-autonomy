# Gazebo worlds

Gazebo Harmonic (`.sdf`) worlds for Swarm Autonomy.

- **`swarm_autonomy_city.sdf`** — the urban test environment: a grid of buildings of varying height with
  street corridors between them, used for navigation, cooperative exploration, and the pursuit
  scenario. Generated by [`gen_city.py`](gen_city.py), which lays out the building grid and writes
  the `.sdf`; edit the generator to change block spacing, building footprint, or height profile.

`run_px4_swarm.sh` syncs the worlds in this directory into the PX4 world path on launch. To use a
PX4 stock world instead, pass its name (e.g. `WORLD=default`).

Keep worlds light enough to run the target vehicle count plus GPU mapping within the available VRAM.
